# BiS-Tooltip WoW 3.3.5 Improved

This is an improved version of the original Bis Tooltip addon for WoW client version 3.3.5.

Original authors: Hellixoid, Dweem, Silver (https://www.curseforge.com/wow/addons/bis-tooltip)
Backport for 3.3.5a by: disruption01 (https://github.com/disruption01/BiS-Tooltip_335a_backport)
Improved by: Spyr581

Data for both vanilla and BC from wowtbc.gg has been added. The choice between addons is made at the bottom of the window. An option has been added to automatically select a given class when selecting a character in the addon; the spec must be selected manually. The ability to see who is suitable for an item, including outside the addon, has been added. Tested on BC; vanilla has not been tested. To save space in the addon window, unnecessary credits have been removed. Maximum functionality, minimum fluff.
